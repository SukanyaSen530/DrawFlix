   # [DrawFlix](https://drawflix.netlify.app/)
     A video library where you can learn to draw.
</div>

#### Stack Used

- React
- SCSS with Bubbles - Component Library
- Javascript

#### Built With -

- Context + useReducer
- React Router v6


# Features

- Home page
- Video listing page
- Filter Videos by Categories
- Search video
- Individual video page
- Like/Unlike a video
- Add/Remove  video to Watch Later
- History management
- Playlist management 
- Authentication: 
    - User Signup 
    - User Login
- Responsive
